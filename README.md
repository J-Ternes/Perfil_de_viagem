# Travelgram ✈️

Projeto fictício de uma página de perfil para uma plataforma de viagens, desenvolvido apenas com **HTML** e **CSS**.

O objetivo deste projeto foi colocar em prática conceitos fundamentais de estruturação de páginas e estilização visual, criando uma interface inspirada em uma rede social de viagens.

![Preview do projeto](assets/images/Imagem%20-%20projeto.png)

## Sobre o projeto

A página apresenta o perfil de uma viajante fictícia, com:

- Logo e menu de navegação;
- Foto e informações do perfil;
- Localização, quantidade de países visitados e fotos;
- Galeria de imagens de destinos turísticos;
- Layout limpo, organizado e focado em fotografia de viagem.

## Tecnologias utilizadas

- HTML5
- CSS3

## Conceitos praticados

Durante o desenvolvimento, foram aplicados conceitos básicos e importantes de HTML e CSS, como:

### HTML

- Uso de tags semânticas para organizar o conteúdo da página;
- Criação de cabeçalho, área de perfil e galeria de fotos;
- Inserção de imagens utilizando a tag `<img>`;
- Estruturação de textos com títulos, parágrafos e listas;
- Uso de links e elementos de navegação.

### CSS

- Estilização de textos, cores e espaçamentos;
- Aplicação de fontes, tamanhos e pesos tipográficos;
- Uso de `display: flex` para alinhar os elementos do cabeçalho e das informações do perfil;
- Uso de `display: grid` ou `flexbox` para construir a galeria de imagens;
- Controle de largura, altura e proporção das imagens;
- Bordas arredondadas com `border-radius`, especialmente na foto de perfil;
- Ajuste de margens, preenchimentos e alinhamento dos componentes;
- Criação de uma interface visualmente equilibrada e organizada.

## Objetivo

Este é um projeto simples de estudo, criado para reforçar os conhecimentos básicos de **HTML** e **CSS** por meio da construção de uma interface semelhante a uma página real de viagens.
